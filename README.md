# Sistema de votação para teste sigProj
### Features: 
- PHP 7.2
- Mysql 8.0
- SGBD PHPMyadmin
- laravel 7
- javascript puro
- jquery
- json

### Funcoes - Admin
1. cadastrar/atualizar/deletar usuarios
2. cadastrar/atualizar/deletar enquetes


### Funcoes - Usuario Comum
1. votar em enquetes
